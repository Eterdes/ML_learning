# Финансовые новости — Анализ тональности для трейдеров 📈🤖

Проект по разработке и оптимизации пайплайна машинного обучения (NLP) для классификации тональности финансовых новостей на три класса: **Negative (0)**, **Neutral (1)** и **Positive (2)**. 

Цель проекта — разработать модель, превосходящую базовое решение (Baseline) по метрике **Macro F1** минимум на 5%.

---

## 📁 Структура проекта

* `analys.py` — основной скрипт проекта (загрузка, EDA, обучение, оценка и инференс).
* `financial_data.csv` — исходный датасет (Financial PhraseBank).
* `requirements.txt` — список необходимых библиотек для работы проекта.
* `best_model.pkl` — сохраненные веса лучшей обученной модели.
* `vectorizer.pkl` — сохраненный векторизатор текстов (TF-IDF).
* `sentiment_distribution.png` — график распределения классов в датасете.
* `text_length_distribution.png` — гистограмма распределения длин текстов.
* `confusion_matrix.png` — матрица ошибок лучшей модели.
* `baseline_result.txt` — зафиксированный результат базовой модели.
* `comparison_table.txt` — таблица сравнения результатов экспериментов.
* `error_analysis.txt` — анализ примеров, на которых модель ошиблась.
